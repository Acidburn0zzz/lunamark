lunamark.writer.html5

HTML 5 writer for lunamark. Extends lunamark.writer.html, but uses <section> tags for sections if options.containers is true.

new(options)

Returns a new HTML 5 writer. options is as in lunamark.writer.html. For a list of fields, see lunamark.writer.generic.
