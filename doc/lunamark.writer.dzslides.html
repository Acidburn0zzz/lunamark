lunamark.writer.dzslides

dzslides writer for lunamark. dzslides is a lightweight HTML5 slide engine by Paul Rouget: https://github.com/paulrouget/dzslides. This writer extends lunamark.writer.html5.

new(options)

Returns a new dzslides writer. options is as in lunamark.writer.html5. However, options.slides is ignored and set to true. For a list of fields, see lunamark.writer.generic.
