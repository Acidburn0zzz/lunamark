lunamark.entities

Functions for dealing with HTML/XML entities.

dec_entity(s)

Given a string of decimal digits, returns a UTF-8 encoded string encoding a unicode character.

hex_entity(s)

Given a string of hexadecimal digits, returns a UTF-8 encoded string encoding a unicode character.

char_entity(s)

Given a character entity name (like ouml), returns a UTF-8 encoded string encoding a unicode character.
